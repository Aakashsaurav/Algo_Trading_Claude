Quick Start Guide

1. Running a Backtest

Enter a symbol (e.g. RELIANCE), choose a strategy, set your dates and capital, then click Run Backtest.

The engine executes signals at next bar's open (no look-ahead bias). Results include: metrics, Streak-style chart, equity curve, and full trade log.

2. Order Types

Market: Execute at next bar's open (default, most realistic).

Limit: Enter only if price pulls back to your limit level (patient entry).

Stop: Enter on breakout above a stop level.

Stop-Loss %: Automatically close a position if it moves against you by X%.

Trailing Stop: Stop level follows price upward, never backward.

3. Parameter Optimizer

Use the Optimizer panel to find the best parameter values. Provide comma-separated values for each parameter (e.g. fast_period → 5,9,13). Use Random Search for large grids (faster). Choose your target metric (Sharpe, Return, etc.).

4. Screener

Scans all symbols with local OHLCV data. Returns today's signals — stocks at buy or sell setups per your strategy. Results are ranked and can be saved as CSV/JSON.

5. Writing Your Own Strategy

from strategies.base import BaseStrategy
from indicators.technical import ema, crossover

class MyStrategy(BaseStrategy):
    def __init__(self, fast=9, slow=21):
        super().__init__(name="My EMA Cross")
        self.fast = fast
        self.slow = slow

    def generate_signals(self, df):
        df = df.copy()
        df['ema_fast'] = ema(df['close'], self.fast)
        df['ema_slow'] = ema(df['close'], self.slow)
        df['signal']   = 0
        df.loc[crossover(df['ema_fast'], df['ema_slow']), 'signal']  = 1
        df.loc[crossunder(df['ema_fast'], df['ema_slow']), 'signal'] = -1
        return df

6. Using External Indicator Libraries

from indicators.bridge import IndicatorBridge

bridge = IndicatorBridge()
print(bridge.library_status())  # See what's installed

# Auto-selects best available library:
rsi  = bridge.rsi(df['close'], 14)
atr  = bridge.atr(df, 14)

# Force a specific library:
ema  = bridge.ema(df['close'], 20, library='pandas_ta')

# Use any pandas-ta indicator by name:
bb   = bridge.pandas_ta_indicator(df, 'bbands', length=20)

# Register your own custom indicator:
bridge.register('my_zscore_rsi',
    lambda c, p: bridge.zscore(bridge.rsi(c, p), 20)
)
